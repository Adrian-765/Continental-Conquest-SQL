Otoño 1915 — Puerto de Venecia, Italia

Tras semanas de navegación por el Mediterráneo, atravesando puertos y costas hostiles, finalmente ponemos pie en Venecia. La ciudad se presenta envuelta en niebla, con el bullicio portuario amortiguado por el silencio de la guerra.

Nuestro próximo objetivo es claro: cruzar hacia Atenas a bordo del SS Eleftheria, un barco de bandera neutral que oficialmente transporta recursos hacia Rumanía. Nadie debe sospechar la verdadera intención.

Pero Nikolas Theron, el capitán del puerto, nos detiene con mirada firme: la logística debe verificarse. Cada contacto en Atenas tiene una capacidad limitada, y cualquier error podría retrasar la operación o comprometer la neutralidad del barco.

Mientras contemplo la bruma que cubre los canales, pienso en los próximos pasos: Atenas, el Bósforo, y luego el corazón del Imperio Ruso.

💾 Consultas SQL de este Nivel
mysql — gran_guerra.sql
1→A · Mayor cap. soldados
SELECT nombre_contacto
FROM puertos_atenas
ORDER BY capacidad_soldados DESC
LIMIT 1;
2→B · Todos los contactos
SELECT nombre_contacto,
       capacidad_soldados,
       capacidad_suministros
FROM puertos_atenas;
3→C · Más de un barco
SELECT nombre_contacto
FROM puertos_atenas
WHERE barcos_disponibles > 1;
4→D · Suministros ≥ 150
SELECT nombre_contacto
FROM puertos_atenas
WHERE capacidad_suministros >= 150;
📋 Estructura de tablas
CREATE TABLE puertos_atenas (
    id_contacto          INT PRIMARY KEY,
    nombre_contacto      VARCHAR(50),
    capacidad_soldados   INT,
    capacidad_suministros INT,
    barcos_disponibles   INT
);
🔍 Análisis Clasificado
ORDER BY capacidad_soldados DESC ordena de mayor a menor. LIMIT 1 devuelve solo el primero. Resultado: Dimitrios Pappas (200 soldados).
Un SELECT sin WHERE ni LIMIT devuelve todos los registros. Útil para ver el panorama completo de los 4 contactos disponibles en Atenas.
Filtra los contactos con más de un barco disponible. Resultado: Theo Marinakis (2 barcos) y Dimitrios Pappas (3 barcos). Importante para asignaciones múltiples.
El operador >= incluye el valor 150 (mayor o igual). Resultado: Theo Marinakis (200) y Dimitrios Pappas (150). Son los que pueden recibir la carga pesada.
