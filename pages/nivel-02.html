Francia nos recibe con un silencio inquietante. Las calles de la costa parecen vacías, como si la guerra misma contuviera la respiración ante nuestro paso. Tras semanas de viaje desde Irlanda, la milicia y yo nos adentramos en territorio enemigo, y cada paso se siente más peligroso que el anterior.

El contacto secreto nos ha guiado hasta el Palacio de Versalles, pero no para admirar su grandeza. Aquí, en los pasillos oscuros y los jardines laberínticos, cada camino serpenteante representa una decisión estratégica.

En el corazón del laberinto se encuentra la mesa de piedra donde nuestro contacto despliega un mapa táctico. Cada ruta, cada bosque, cada río está marcado; cada ciudad es un punto de riesgo.

El laberinto es un espejo de la guerra misma: complejo, peligroso, y lleno de caminos que pueden llevar tanto a la victoria como a la derrota.

A · Unidades listas — JOIN
SELECT u.id_unidad, tropas, municion
FROM unidades u
JOIN suministros s USING(id_unidad);
B · Sin munición — LEFT JOIN
SELECT u.id_unidad
FROM unidades u
LEFT JOIN suministros s USING(id_unidad)
WHERE s.id_unidad IS NULL;
C · Tropas por país — GROUP BY
SELECT pais_origen, SUM(tropas)
FROM unidades
GROUP BY pais_origen;
D · Rutas seguras — WHERE
SELECT *
FROM rutas
WHERE nivel_riesgo < 3;
📋 Estructura de tablas
CREATE TABLE unidades (
    id_unidad   INT PRIMARY KEY,
    nombre      VARCHAR(50),
    pais_origen VARCHAR(50),
    tipo        VARCHAR(30),
    tropas      INT
);
CREATE TABLE suministros (
    id_suministro      INT PRIMARY KEY,
    id_unidad          INT,
    alimento           INT,
    municion           INT,
    descuento_logistico DECIMAL(5,2)
);
CREATE TABLE rutas (
    id_ruta     INT PRIMARY KEY,
    origen      VARCHAR(50),
    destino     VARCHAR(50),
    nivel_riesgo INT
);
🔍 Análisis Clasificado
JOIN une unidades con suministros usando el campo id_unidad compartido (USING). Solo devuelve unidades que TIENEN suministros asignados.
LEFT JOIN devuelve TODAS las unidades, aunque no tengan suministros. WHERE s.id_unidad IS NULL filtra solo las que NO tienen munición asignada — las más vulnerables.
GROUP BY agrupa todas las filas con el mismo país de origen y SUM(tropas) suma sus efectivos. Permite ver de qué país proviene más fuerza militar.
El operador < filtra solo las rutas con valor numérico inferior a 3. Valores más bajos = menos riesgo. Son las rutas seguras por las que debe moverse la milicia.
