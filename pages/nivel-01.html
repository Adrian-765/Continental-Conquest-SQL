3 de abril de 1915 — A bordo rumbo a Irlanda

El mar está tranquilo, pero no hay paz en mi cabeza. Las olas golpean el casco del barco con un ritmo monótono, y cada golpe parece recordarme la magnitud de lo que está por venir. Hoy dejamos España, nuestras costas familiares, y nos dirigimos hacia Irlanda. No es un simple viaje: debo ganarme la confianza de John Redmond, el hombre que puede inclinar la balanza de toda Europa.

Redmond no duda del ideal irlandés. Duda de Europa. Sabe que si las Potencias Centrales y el Imperio Austrohúngaro caen, el Imperio Ruso se alzará con una fuerza capaz de rivalizar con los grandes de Occidente. No entregará los Irish Volunteers a una misión mal planificada.

—"Demostradme que sabéis equilibrar fuerzas. Que comprendéis el peso real de cada aliado. Que no improvisáis."

El peso de la responsabilidad es insoportable a veces. Cada decisión que tome hoy podría determinar la vida de cientos de hombres y el destino de naciones enteras. Pero siento una llama dentro de mí: determinación pura.

Simular el nuevo equilibrio de tropas si Irlanda se suma a la coalición, comprobando que el total supere el umbral estratégico mínimo.

💾 Consultas SQL de este Nivel

A · Fuerza total de la coalición
SELECT SUM(tropas)       AS total_tropas,
       SUM(suministros) AS total_suministros
FROM aliados;
B · Voluntarios por región
SELECT region,
       SUM(efectivos) AS total_region
FROM voluntarios_irlandeses
GROUP BY region;
C · Frentes por presión militar
SELECT nombre,
       presion_militar
FROM frentes_europeos
ORDER BY presion_militar DESC;
D · Nuevo total con Irlanda
SELECT
  (SELECT SUM(tropas)    FROM aliados) +
  (SELECT SUM(efectivos) FROM voluntarios_irlandeses)
  AS nuevo_total_coalicion;
📋 Estructura de tablas
CREATE TABLE aliados (
    id_aliado  INT PRIMARY KEY,
    pais       VARCHAR(50),
    tropas     INT,
    suministros INT
);
CREATE TABLE voluntarios_irlandeses (
    id_voluntario INT PRIMARY KEY,
    region        VARCHAR(50),
    efectivos     INT
);
CREATE TABLE frentes_europeos (
    id_frente       INT PRIMARY KEY,
    nombre          VARCHAR(50),
    presion_militar INT
);
🔍 Análisis Clasificado
Usa SUM() dos veces en el mismo SELECT para obtener el total de tropas y suministros a la vez. Ambos valores deben superar los umbrales mínimos que Redmond exige.
GROUP BY region agrupa los voluntarios por zona geográfica irlandesa. SUM(efectivos) calcula cuántos hombres aporta cada región. Redmond quiere ver distribución territorial, no solo totales.
ORDER BY presion_militar DESC ordena los frentes del más presionado al menos. Esto justifica por qué es urgente abrir un nuevo frente en el este contra el Imperio Ruso.
La consulta D usa dos subconsultas dentro de un SELECT para sumar tropas aliadas + efectivos irlandeses en un solo resultado. Si el total supera el umbral, Irlanda se une.
