No querían pruebas. Querían contradicciones.

💾 Consultas SQL
norte_n6.sql
1 · Pasajeros sensibles — WHERE
SELECT nombre
FROM pasajeros
WHERE nacionalidad = 'Irlandesa';
2 · Coherencia manifiesto/ruta — JOIN
SELECT m.destino_declarado,
       r.puerto_siguiente
FROM manifiestos m
JOIN rutas_registradas r USING(id_barco)
WHERE m.destino_declarado <> r.puerto_siguiente;
3 · Pasajeros por nacionalidad
SELECT nacionalidad,
       COUNT(*) total
FROM pasajeros
GROUP BY nacionalidad;
4 · Vista de coherencia total
CREATE VIEW control_britanico AS
SELECT m.id_barco
FROM manifiestos m
JOIN rutas_registradas r USING(id_barco)
WHERE m.destino_declarado = r.puerto_siguiente;

-- Si SELECT * FROM control_britanico
-- devuelve el barco → AUTORIZADO
-- Si está vacía → RETENIDO
📋 Estructura de tablas
CREATE TABLE pasajeros (
    id_pasajero  INT PRIMARY KEY,
    nombre       VARCHAR(50),
    nacionalidad VARCHAR(30)
);
CREATE TABLE manifiestos (
    id_barco          INT,
    destino_declarado VARCHAR(50),
    tipo_carga        VARCHAR(30)
);
CREATE TABLE rutas_registradas (
    id_barco        INT,
    puerto_anterior VARCHAR(50),
    puerto_siguiente VARCHAR(50)
);
🔍 Análisis Táctico
La nacionalidad irlandesa en 1915 era políticamente sensible: el Alzamiento de Pascua se estaba preparando. WHERE nacionalidad='Irlandesa' devuelve a Seamus O'Callaghan como pasajero de riesgo inmediato para el Almirantazgo.
JOIN une manifiestos con rutas_registradas por id_barco. WHERE destino_declarado <> puerto_siguiente busca contradicciones. En este caso ambos son 'Bergen', así que el resultado está vacío — coherencia total, no hay contradicciones.
CREATE VIEW no almacena datos: define una consulta reutilizable. Después de crearla, SELECT * FROM control_britanico ejecuta la consulta. Si devuelve el id=1, el barco tiene coherencia y puede salir. Si está vacía, queda retenido.
GROUP BY nacionalidad + COUNT(*) muestra: Irlandesa (1), Noruega (1), Alemana (1). Una tripulación con ciudadano alemán en 1915 resulta muy sospechosa para el Almirantazgo Británico aunque no genere error lógico.
